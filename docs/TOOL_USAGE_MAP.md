# Tool Usage Map

## Advisory knowledge
Use Confluence tools for:
- architecture guidance
- pattern selection guidance
- naming conventions
- FAQs
- troubleshooting playbooks

## Runtime truth
Use Databricks tools for:
- existing jobs
- notebook source
- cluster state
- run history
- SQL validation
- Python validation
- table profiling
- artifact mapping
- runtime diagnostics

## Local implementation
Use workspace tools for:
- source code
- tests
- manifests
- prompts
- agents
- generated local artifacts
